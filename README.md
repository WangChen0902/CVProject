## 测试模型 ##
1. 将待测试的图片放入Test目录中
2. 进入KinD目录，运行python evaluate.py，结果将存入主目录下的Pic目录中。
3. 进入yolo3目录，运行python new_my_yolo_video.py --image，结果将存入主目录下的Result目录中。
